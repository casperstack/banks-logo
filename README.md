# โลโก้ธนาคาร

## รายชื่อข้อมูลธนาคารประเทศไทย

- Fork มาจาก https://github.com/omise/banks-logo
- ได้มีการปรับเพิ่ม NodeJS สำหรับการติดตั้งบน NPM
- ลบธนาคารบางส่วนออก

## รายละเอียด
- โลโก้แบบ SVG
- Bank Code
- ชื่อย่อธนาคาร
- ชื่อทางการ
- ชื่อภาษาอังกฤษ
- ชื่อภาษาไทย
